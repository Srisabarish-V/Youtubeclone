# Youtubeclone



A **basic YouTube clone** built with **React**, replicating the core functionality of YouTube such as video listing, video playback, and a simple navigation bar.

---

## Features

- Display a list of videos with thumbnails and titles  
- Play selected videos  
- Simple navigation sidebar (Home, Explore, Subscriptions, Library)  
- Responsive layout  
- Built using **React**, **CSS**, and **Vite**  

---

## Tech Stack

- **Frontend:** React, JSX, CSS  
- **Bundler:** Vite  
- **Icons & Assets:** Local images/icons for UI elements
